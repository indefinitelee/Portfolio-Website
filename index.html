Fantasy Premiere League Stats Tracker

This project analyzes Barclay's Premiere League player statistics to provide insights to the more than 4 million fantasy premiere league players. More info ...

The project uses a PERN stack to retrieve and parse the data. Player performance graphs were implemented with the Chartjs library.

Serendipity

Serendipity helps you make last minute decisions by searching for activities based on your location. More info ...

Serendipity uses the HTML5 geolocation feature to retrieve your location and then searches the Ticket Master API to show you nearby events. When you click on an event it searches the YouTube API to return relevant videos.

This Day in History

This day in history allows users to search the New York Times' API and retrieve the top articles for a given date. More info ...

The app uses a MongoDB backend to allow users to sign-up, login, and save links to articles.
